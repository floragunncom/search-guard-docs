# Get all trust stores
{: .no_toc}

{% include toc.md %}


```
GET /_signals/truststores/
```

Load information about all trust stores 


## Responses

### 200 OK

List which contains all defined trust stores in the system.

### 403 Forbidden

The user does not have the required permissions to access the endpoint.

## Permissions

For being able to access the endpoint, the user needs to have the privilege `cluster:admin:searchguard:signals:truststores/findall`.

## Examples

### Get all trust stores

```
GET /_signals/truststores/
```

**Response**

```
200 OK
```
