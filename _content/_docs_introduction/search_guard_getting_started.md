# Getting Started with Search Guard

Welcome to Search Guard, the enterprise security suite for Elasticsearch. This section provides everything you need to get started with Search Guard, from understanding core concepts to running your first secure Elasticsearch cluster.

## What You'll Learn

In this section, you'll find:

- **[What is Search Guard?](security-for-elasticsearch)** - Overview of Search Guard's features and capabilities
- **[Key Concepts](main-concepts)** - Core security concepts including authentication, authorization, roles, and permissions
- **[Quickstart](search-guard-quickstart)** - Fast-track guides to get Search Guard running quickly
- **[Editions and Licensing](search-guard-editions)** - Understand the different Search Guard editions and licensing options

## Choose Your Path

### New to Search Guard?

Start with [What is Search Guard?](security-for-elasticsearch) to understand the security features available, then review the [Key Concepts](main-concepts) to learn the fundamental security model.

### Ready to Install?

Jump directly to the [Quickstart](search-guard-quickstart) guide. Choose between:
- **[Demo Installer](demo-installer)** - Fastest way to get started (recommended for evaluation)
- **[Manual Installation](manual-installation)** - Step-by-step installation for custom setups

### Planning a Deployment?

Review the [Editions and Licensing](search-guard-editions) section to understand which features are available in each edition and choose the right license for your needs.

## Next Steps

After completing this section, proceed to [Installation and Setup](search-guard-installation-setup) to install Search Guard in your environment.
